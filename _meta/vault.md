## Note types

| Type | Purpose | Typical folder |
|---|---|---|
| Literature note | Source-faithful representation of a single source | `references/` |
| Permanent note | Atomic, link-dense, integrated into the zettelkasten | `main/` |
| Fleeting note | Transient capture, lives in daily notes | `daily/` |
| Index note | Hub for a topic; links to related notes | `index/` or root |
